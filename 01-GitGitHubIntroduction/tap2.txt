20 linux commands


ls: Lists directory contents.

cd: Changes the current directory.

pwd: Prints the current working directory.

mkdir: Creates a new directory.

rm: Removes files or directories.

cp: Copies files or directories.

mv: Moves or renames files or directories.

touch: Creates an empty file or updates the timestamp of an existing file.

cat: Concatenates and displays the contents of files.

grep: Searches for patterns in files.

find: Searches for files and directories in a directory hierarchy.

chmod: Changes file permissions.

chown: Changes file ownership.

sudo: Executes a command with superuser (administrative) privileges.

tar: Archives files and directories.

gzip: Compresses files.

wget: Downloads files from the internet.

ssh: Connects to a remote machine over a secure shell.

top: Displays dynamic real-time information about running processes.

df: Shows disk space usage of file systems.
